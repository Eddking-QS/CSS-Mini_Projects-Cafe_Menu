# HTML Mini Project- Café Menu

## Description

A simple menu page for the Grey House Café made with HTML and CSS.

## Features

**HTML**

-   Headings
-   Paragraphs
-   Images
-   Links
-   Sections
-   Articles

**CSS**

-   Background Image
-   Text Alignment
-   Font Styling
-   Margins and Padding
-   Display Properties
